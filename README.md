# Contador de Interrupções

Projeto simples em HTML, CSS e JavaScript para contar interrupções durante a aula de uma pessoa "chata".

## O que faz

- inicia e encerra cada interrupção;
- registra início, fim e duração no histórico;
- gera um arquivo CSV com os dados registrados.

## Como usar

1. Abra o `index.html` no navegador.
2. Clique em `Iniciar interrupção` quando a interrupção começar.
3. Clique em `Encerrar interrupção` quando ela terminar.
4. Clique em `Gerar CSV` para baixar os dados.
